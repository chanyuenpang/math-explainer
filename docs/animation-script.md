# MathExplainer 动画脚本 - math-001

**题目**：四边形 ABCD，∠A=∠BCD=90°，BC=DC，DE=AB，E 在 AD 延长线上
**目标**：证明 (1) ∠ABC=∠EDC；(2) △ABC≌△EDC

---

## 动画步骤设计

### 步骤 0：开场 - 图形绘制
- 顺序绘制：A → B → C → D → A → 延伸至 E
- 动画效果：逐条边淡入绘制，每条边 0.3s
- 标注：∠A 直角、∠BCD 直角
- 标注：BC = DC，AB = DE（DE 后面单独绘制）

### 步骤 1：分析已知条件
- 高亮 ∠A（蓝色脉冲），文字："∠A = 90°"
- 高亮 ∠BCD（蓝色脉冲），文字："∠BCD = 90°"
- 同时高亮 BC 和 DC，文字："BC = DC"
- 显示："四边形 ABCD 中，∠A 和 ∠BCD 都是直角"

### 步骤 2：观察图形结构
- 延伸 AD 到 E，DE 线段动画绘制（虚线→实线）
- 高亮 AB 和 DE，文字："DE = AB"
- 标注："E 是 AD 延长线上的点，A-D-E 共线"

### 步骤 3：证明 ∠ABC = ∠EDC（第一问）
- 在 ∠A 处画弧线标注 90°
- 在 ∠BCD 处画弧线标注 90°
- 文字推导："∵ ∠A + ∠BCD = 180°"
- "∵ A、D、E 共线"
- "∴ ∠ADC = 90°"
- 高亮 ∠ABC 和 ∠EDC（两角都用橙色高亮）
- 显示结论："∴ ∠ABC = ∠EDC" ✓

### 步骤 4：准备全等证明
- △ABC 填充红色半透明（rgba(239,68,68,0.2)）
- △EDC 填充绿色半透明（rgba(16,185,129,0.2)）
- 两三角形分别高亮边框

### 步骤 5：验证 SAS 全等条件
- **条件 1**：高亮 AB + DE，文字："AB = DE（已知）"，打勾 ✓
- **条件 2**：高亮 BC + DC，文字："BC = DC（已知）"，打勾 ✓
- **条件 3**：高亮 ∠A + ∠EDC，文字："∠A = ∠EDC = 90°"，打勾 ✓

### 步骤 6：全等动画 + 结论
- △ABC 边缘高亮放大
- 动画：△ABC 旋转 180°（或平移）到 △EDC 位置
- 与 △EDC 完全重合
- 显示："∴ △ABC ≌ △EDC（SAS）" ✓
- 显示："证毕" 标记

---

## 技术实现要点

1. **SVG 绘制**：所有图形用 SVG 实现
2. **GSAP 动画**：
   - 边绘制：stroke-dashoffset 动画
   - 高亮：stroke + strokeWidth 动画 + yoyo 闪烁
   - 填充：fill-opacity 动画
   - 三角形变换：transform: rotate/translate 动画
3. **文字说明**：每步同步显示在右侧
4. **控制**：上一步/下一步按钮，点击后触发对应动画

## 文件结构

- `GeometryCanvas.tsx` - 核心 SVG 画布组件
- `math-001.json` - 题目数据 + 步骤动画配置
- 动画配置用 stepAnimations 字段定义每步的高亮、绘制、变换
